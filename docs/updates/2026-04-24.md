  - このbooleanフィールドは、ユーザーレベルのレポートにおいて既存のデータを反映。
  - 詳細は[こちら](https://github.blog/changelog/2026-04-23-copilot-cloud-agent-fields-added-to-usage-metrics)。

- **Web上でのデバッグ改善**
  - GitHub Copilot Chatが、stack traceを貼り付けることでエラーの根本原因をより迅速に特定。
  - Copilotがstack traceをより信頼性高く認識するようになった。
  - 詳細は[こちら](https://github.blog/changelog/2026-04-23-better-debugging-with-github-copilot-on-the-web)。

- **IssuesおよびProjectsからエージェントセッションの管理**
  - IssuesやProjectsから直接クラウドエージェントセッションを表示・操作可能に。
  - ワークフローを離れずにエージェントの活動をより良く把握できる。
  - 詳細は[こちら](https://github.blog/changelog/2026-04-23-view-and-manage-agent-sessions-from-issues-and-projects)。

- **GitHub Copilot Businessの新規セルフサーブサインアップ一時停止**

  - 詳細は[こちら](https://github.blog/changelog/2026-04-22-pausing-new-self-serve-signups-for-github-copilot-business)。
