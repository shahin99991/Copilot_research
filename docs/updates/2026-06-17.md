- **Power Fx: User Defined Types (UDTs)**:

- **Power Apps MCP Server**:
  - エンタープライズエージェント向けに、閉ループ学習機能が導入。
  - 修正がエージェントを自動的にビジネスに適応させる。

- **Agentic Administration**:
  - Dataverse管理者スキル（dv-admin）が一般提供のパブリックプレビューに登場。
  - 自然言語を使用してDataverse環境を大規模に管理可能。

- **Dataverse as Agent Data Platform**:
  - Microsoft Dataverseはエージェントデータプラットフォームとして、データアクセスとビジネス理解を提供。

- **Business Skills for Agents**:
  - Dataverse内で、組織のプロセスやポリシーを自然言語の指示としてキャプチャし、AIエージェントが学習できるようにする機能が導入。
